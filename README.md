## ✨ Features & Architecture

* **Modern Dark Theme:** Styled with a premium dark blue color scheme (`#081b29` and `#112e42`) and vibrant cyan highlights (`#00abf0`).
* **Non-Blocking Top Notice Banner:** Floats at the top of the viewport on page load. It dynamically offsets the sticky header and main content layout to prevent overlap and slides out of view smoothly when "I Understand" is clicked.
* **Animated Typing Subheading:** Displays active changing roles ("2nd year Computer Engineering Student" and "Full-Stack Web Development Enthusiast") using dynamic JavaScript DOM text switching and CSS typing animations.
* **Scroll-Triggered Entry Animations:** Content boxes, images, lists, and header navigation elements slide and fade in gracefully as the user scrolls down the page.
* **Active Navigation State Tracking:** The navigation menu items automatically highlight themselves based on the current viewport scroll position.
* **Responsive Layouts:** Crafted with CSS Media Queries for a seamless viewing experience across desktops, tablets, and mobile devices.
* **Formspree Contact Integration:** Fully operational contact form linking inputs directly to Formspree for backend processing and notification delivery.

## 🚀 Running Locally

To clone and run this portfolio website on your local machine:

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/pjha91275/Prince_Jha_Portfolio.git
   cd Prince_Jha_Portfolio
   ```
2. **Open in Browser:**
   Simply double-click `index.html` or run a local static server to test:
   * **Node (http-server):** `npx http-server`
   * **VS Code Live Server:** Right-click `index.html` and click *Open with Live Server*.
